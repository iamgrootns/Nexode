Nexode CLI po defaultu pokreće [TUI](/docs/tui) kada se pokrene bez ikakvih argumenata.

```bash
nexode
```

Ali takođe prihvata naredbe kao što je dokumentovano na ovoj stranici. Ovo vam omogućava programsku interakciju sa Nexode.

```bash
nexode run "Explain how closures work in JavaScript"
```

---

### tui

Pokrenite Nexode terminalski korisnički interfejs.

```bash
nexode [project]
```

#### Opcije

| Opcija       | Kratko | Opis                                                                     |
| ------------ | ------ | ------------------------------------------------------------------------ |
| `--continue` | `-c`   | Nastavite posljednju sesiju                                              |
| `--session`  | `-s`   | ID sesije za nastavak                                                    |
| `--fork`     |        | Forkujte sesiju pri nastavku (koristiti sa `--continue` ili `--session`) |
| `--prompt`   |        | Prompt za upotrebu                                                       |
| `--model`    | `-m`   | Model za korištenje u obliku provider/model                              |
| `--agent`    |        | Agent za korištenje                                                      |
| `--port`     |        | Port na kojem treba slušati                                              |
| `--hostname` |        | Hostname na kojem treba slušati                                          |

## Naredbe

Nexode CLI takođe ima sljedeće naredbe.

---

### agent

Upravljajte Nexode agentima.

```bash
nexode agent [command]
```

---

### attach

Priključite terminal na već pokrenut Nexode backend server pokrenut putem `serve` ili `web` naredbi.

```bash
nexode attach [url]
```

Ovo omogućava korištenje TUI-ja sa udaljenim Nexode backend-om. Na primjer:

```bash
# Start the backend server for web/mobile access
nexode web --port 4096 --hostname 0.0.0.0

# In another terminal, attach the TUI to the running backend
nexode attach http://10.20.30.40:4096
```

#### Opcije

| Opcija      | Kratko | Opis                                 |
| ----------- | ------ | ------------------------------------ |
| `--dir`     |        | Radni direktorij za pokretanje TUI-a |
| `--session` | `-s`   | ID sesije za nastavak                |

---

#### create

Kreirajte novog agenta s prilagođenom konfiguracijom.

```bash
nexode agent create
```

Ova naredba će vas voditi kroz kreiranje novog agenta sa prilagođenim sistemskim promptom i konfiguracijom alata.

---

#### list

Navedite sve dostupne agente.

```bash
nexode agent list
```

---

### auth

Naredba za upravljanje vjerodajnicama i prijavom za provajdere.

```bash
nexode auth [command]
```

---

#### login

Nexode pokreće lista provajdera na [Models.dev](https://models.dev), tako da možete koristiti `nexode auth login` da konfigurirate API ključeve za bilo kojeg provajdera kojeg želite koristiti. Ovo je pohranjeno u `~/.local/share/nexode/auth.json`.

```bash
nexode auth login
```

Kada se Nexode pokrene, učitava dobavljače iz datoteke vjerodajnica. I ako postoje neki ključevi definirani u vašim okruženjima ili `.env` fajl u vašem projektu.

---

#### list

Navodi sve autentifikovane dobavljače pohranjene u datoteci vjerodajnica.

```bash
nexode auth list
```

Ili kratka verzija.

```bash
nexode auth ls
```

---

#### logout

Odjavljuje vas s provajdera tako što ga briše iz datoteke vjerodajnica.

```bash
nexode auth logout
```

### github


---

#### install

Instalirajte GitHub agenta u svoj repozitorij.

Ovo postavlja neophodni tok rada GitHub Actions i vodi vas kroz proces konfiguracije. [Saznajte više](/docs/github).

### mcp

Upravljajte Model Context Protocol (MCP) serverima.

```bash
nexode mcp [command]
```

---

#### add

Dodajte MCP server svojoj konfiguraciji.

```bash
nexode mcp add
```

Ova naredba će vas voditi kroz dodavanje lokalnog ili udaljenog MCP servera.

---

#### list

Navedite sve konfigurirane MCP servere i njihov status veze.

```bash
nexode mcp list
```

Ili koristite kratku verziju.

```bash
nexode mcp ls
```

---

#### auth

Autentifikujte se sa MCP serverom koji je omogućen za OAuth.

```bash
nexode mcp auth [name]
```

Ako ne navedete ime servera, od vas će biti zatraženo da izaberete neki od dostupnih servera koji podržavaju OAuth.
Također možete navesti servere koji podržavaju OAuth i njihov status autentifikacije.

```bash
nexode mcp auth list
```

Ili koristite kratku verziju.

```bash
nexode mcp auth ls
```

---

#### logout

Uklonite OAuth vjerodajnice za MCP server.

```bash
nexode mcp logout [name]
```

---

#### debug

Otklanjanje grešaka (debug) OAuth veze sa MCP serverom.

```bash
nexode mcp debug <name>
```

### models

Navedite sve dostupne modele konfiguriranih provajdera.

```bash
nexode models [provider]
```

Ova naredba prikazuje sve modele dostupne kod vaših konfiguriranih provajdera u formatu `provider/model`.
Ovo je korisno za pronalaženje tačnog naziva modela za korištenje u [vašoj konfiguraciji](/docs/config/).
Opciono možete proslijediti ID provajdera za filtriranje modela po tom dobavljaču.

```bash
nexode models anthropic
```

#### Opcije

| Opcija      | Opis                                                                     |
| ----------- | ------------------------------------------------------------------------ |
| `--refresh` | Osvježite keš modela sa models.dev                                       |
| `--verbose` | Koristite detaljniji izlaz modela (uključuje metapodatke poput troškova) |

Koristite `--refresh` zastavicu da ažurirate keširanu listu modela. Ovo je korisno kada su novi modeli dodani provajderu i želite da ih vidite u Nexode.

```bash
nexode models --refresh
```

### run

Pokrenite Nexode u neinteraktivnom modu tako što ćete direktno proslijediti prompt.

```bash
nexode run [message..]
```

Ovo je korisno za skriptiranje, automatizaciju ili kada želite brz odgovor bez pokretanja punog TUI-ja. Na primjer:

```bash "nexode run"
nexode run Explain the use of context in Go
```

Također možete priključiti pokrenutu `nexode serve` instancu kako biste izbjegli vrijeme hladnog pokretanja MCP servera pri svakom pokretanju:

```bash
# Start a headless server in one terminal
nexode serve

# In another terminal, run commands that attach to it
nexode run --attach http://localhost:4096 "Explain async/await in JavaScript"
```

#### Opcije

| Opcija       | Kratko | Opis                                                                     |
| ------------ | ------ | ------------------------------------------------------------------------ |
| `--command`  |        | Naredba za pokretanje, koristite poruku za argumente                     |
| `--continue` | `-c`   | Nastavite posljednju sesiju                                              |
| `--session`  | `-s`   | ID sesije za nastavak                                                    |
| `--fork`     |        | Forkujte sesiju pri nastavku (koristiti sa `--continue` ili `--session`) |
| `--share`    |        | Podijelite sesiju                                                        |
| `--model`    | `-m`   | Model za korištenje u obliku provider/model                              |
| `--agent`    |        | Agent za korištenje                                                      |
| `--file`     | `-f`   | Fajlovi koje treba priložiti poruci                                      |
| `--format`   |        | Format: default (formatiran) ili json (sirovi JSON događaji)             |
| `--title`    |        | Naslov sesije (koristi skraćeni prompt ako nije navedena vrijednost)     |
| `--attach`   |        | Priključite na pokrenuti Nexode server (npr. http://localhost:4096)    |
| `--port`     |        | Port za lokalni server (zadano na nasumični port)                        |

### serve

Pokrenite Nexode headless server za API pristup. Pogledajte [server docs](/docs/server) za kompletan HTTP interfejs.

```bash
nexode serve
```

Ovo pokreće HTTP server koji pruža API pristup funkcionalnosti Nexode-a bez TUI interfejsa. Postavite `NEXODE_SERVER_PASSWORD` da omogućite HTTP osnovnu auth (korisničko ime je zadano na `nexode`).

#### Opcije

| Opcija       | Opis                                                  |
| ------------ | ----------------------------------------------------- |
| `--port`     | Port na kojem treba slušati                           |
| `--hostname` | Hostname na kojem treba slušati                       |
| `--mdns`     | Omogući mDNS otkrivanje                               |
| `--cors`     | Dodatni origin(i) pretraživača koji dozvoljavaju CORS |

### stats

Prikaži statistiku upotrebe tokena i troškova za vaše Nexode sesije.

```bash
nexode stats
```

#### Opcije

| Opcija      | Opis                                                                                                       |
| ----------- | ---------------------------------------------------------------------------------------------------------- |
| `--days`    | Prikaži statistiku za zadnjih N dana (sva vremena)                                                         |
| `--tools`   | Broj alata za prikaz (svi)                                                                                 |
| `--models`  | Prikaži raščlambu korištenja modela (skriveno prema zadanim postavkama). Proslijedite broj za prikaz top N |
| `--project` | Filtriraj po projektu (svi projekti, prazan niz: trenutni projekt)                                         |

---

### export

Izvezite podatke sesije kao JSON.

```bash
nexode export [sessionID]
```

Ako ne unesete ID sesije, od vas će biti zatraženo da odaberete neku od dostupnih sesija.

---

### import

Uvezite podatke sesije iz JSON datoteke ili Nexode dijeljenog URL-a.

```bash
nexode import <file>
```

Možete uvesti iz lokalne datoteke ili Nexode dijeljenog URL-a.

```bash
nexode import session.json
nexode import https://opncd.ai/s/abc123
```

### web

Pokrenite Nexode headless server sa web interfejsom.

```bash
nexode web
```

Ovo pokreće HTTP server i otvara web pretraživač za pristup Nexode-u preko web interfejsa. Postavite `NEXODE_SERVER_PASSWORD` da omogućite HTTP osnovnu auth (korisničko ime je zadano na `nexode`).

#### Opcije

| Opcija       | Opis                                                  |
| ------------ | ----------------------------------------------------- |
| `--port`     | Port na kojem treba slušati                           |
| `--hostname` | Hostname na kojem treba slušati                       |
| `--mdns`     | Omogući mDNS otkrivanje                               |
| `--cors`     | Dodatni origin(i) pretraživača koji dozvoljavaju CORS |

### acp

Pokrenite ACP (Agent Client Protocol) server.

```bash
nexode acp
```

Ova naredba pokreće ACP server koji komunicira preko stdin/stdout koristeći nd-JSON.

#### Opcije

| Opcija       | Opis                        |
| ------------ | --------------------------- |
| `--cwd`      | Radni direktorij            |
| `--port`     | Port na kojem treba slušati |
| `--hostname` | Hostname na kojem slušati   |

---

### uninstall

Deinstalirajte Nexode i uklonite sve povezane datoteke.

```bash
nexode uninstall
```

#### Opcije

| Opcija          | Kratko | Opis                                          |
| --------------- | ------ | --------------------------------------------- |
| `--keep-config` | `-c`   | Sačuvajte konfiguracijske datoteke            |
| `--keep-data`   | `-d`   | Sačuvajte podatke i snimke sesije             |
| `--dry-run`     |        | Pokažite šta bi bilo uklonjeno bez uklanjanja |
| `--force`       | `-f`   | Preskoči upite za potvrdu                     |

---

### upgrade

Ažurira Nexode na najnoviju verziju ili određenu verziju.

```bash
nexode upgrade [target]
```

Za nadogradnju na najnoviju verziju.

```bash
nexode upgrade
```

Za nadogradnju na određenu verziju.

```bash
nexode upgrade v0.1.48
```

#### Opcije

| Opcija     | Kratko | Opis                                                    |
| ---------- | ------ | ------------------------------------------------------- |
| `--method` | `-m`   | Korišteni način instalacije; curl, npm, pnpm, bun, brew |

## Varijable okruženja

Nexode se može konfigurirati pomoću varijabli okruženja.

| Varijabla                             | Tip     | Opis                                                               |
| ------------------------------------- | ------- | ------------------------------------------------------------------ |
| `NEXODE_AUTO_SHARE`                 | boolean | Automatski dijeli sesije                                           |
| `NEXODE_GIT_BASH_PATH`              | string  | Putanja do Git Bash izvršne datoteke na Windows-u                  |
| `NEXODE_CONFIG`                     | string  | Putanja do konfiguracijskog fajla                                  |
| `NEXODE_TUI_CONFIG`                 | string  | Putanja do TUI konfiguracijskog fajla                              |
| `NEXODE_CONFIG_DIR`                 | string  | Putanja do konfiguracijskog direktorija                            |
| `NEXODE_CONFIG_CONTENT`             | string  | Inline json konfiguracijski sadržaj                                |
| `NEXODE_DISABLE_AUTOUPDATE`         | boolean | Onemogući automatske provjere ažuriranja                           |
| `NEXODE_DISABLE_PRUNE`              | boolean | Onemogući brisanje (pruning) starih podataka                       |
| `NEXODE_DISABLE_TERMINAL_TITLE`     | boolean | Onemogući automatsko ažuriranje naslova terminala                  |
| `NEXODE_PERMISSION`                 | string  | Inline json konfiguracija dozvola                                  |
| `NEXODE_DISABLE_DEFAULT_PLUGINS`    | boolean | Onemogući podrazumijevane dodatke (plugins)                        |
| `NEXODE_DISABLE_LSP_DOWNLOAD`       | boolean | Onemogući automatsko preuzimanje LSP servera                       |
| `NEXODE_ENABLE_EXPERIMENTAL_MODELS` | boolean | Omogući eksperimentalne modele                                     |
| `NEXODE_DISABLE_AUTOCOMPACT`        | boolean | Onemogući automatsko sažimanje konteksta                           |
| `NEXODE_DISABLE_CLAUDE_CODE`        | boolean | Onemogući čitanje iz `.claude` (prompt + vještine)                 |
| `NEXODE_DISABLE_CLAUDE_CODE_PROMPT` | boolean | Onemogući čitanje `~/.claude/CLAUDE.md`                            |
| `NEXODE_DISABLE_CLAUDE_CODE_SKILLS` | boolean | Onemogući učitavanje `.claude/skills`                              |
| `NEXODE_DISABLE_MODELS_FETCH`       | boolean | Onemogući dohvaćanje modela iz udaljenih izvora                    |
| `NEXODE_FAKE_VCS`                   | string  | Lažni VCS provajder za potrebe testiranja                          |
| `NEXODE_DISABLE_FILETIME_CHECK`     | boolean | Onemogući provjeru vremena datoteke radi optimizacije              |
| `NEXODE_CLIENT`                     | string  | Identifikator klijenta (zadano na `cli`)                           |
| `NEXODE_ENABLE_EXA`                 | boolean | Omogući Exa alate za web pretraživanje                             |
| `NEXODE_SERVER_PASSWORD`            | string  | Omogući osnovnu autentifikaciju za `serve`/`web`                   |
| `NEXODE_SERVER_USERNAME`            | string  | Poništi osnovno korisničko ime autentifikacije (zadano `nexode`) |
| `NEXODE_MODELS_URL`                 | string  | Prilagođeni URL za dohvaćanje konfiguracije modela                 |
