ชุดโครงการชุมชนที่สร้างขึ้นบน Nexode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Nexode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-nexode](https://github.com/awesome-nexode/awesome-nexode) และ [nexode.cafe](https://nexode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
