| [nexode-skillful](https://github.com/zenobi-us/nexode-skillful)                                       | Nexode エージェントがスキルの検出と挿入を使用してオンデマンドでプロンプトを遅延ロードできるようにする              |
| [nexode-supermemory](https://github.com/supermemoryai/nexode-supermemory)                             | スーパーメモリを使用したセッション間での永続メモリ                                                                   |
| [@plannotator/nexode](https://github.com/backnotprop/plannotator/tree/main/apps/nexode-plugin)        | 視覚的な注釈とプライベート/オフライン共有による対話型の計画レビュー                                                  |
| [@openspoon/subtask2](https://github.com/spoons-and-mirrors/subtask2)                                     | Nexode/コマンドをきめ細かいフロー制御を備えた強力なオーケストレーションシステムに拡張                              |
| [nexode-scheduler](https://github.com/different-ai/nexode-scheduler)                                  | launchd (Mac) または systemd (Linux) を cron 構文で使用して、定期的なジョブをスケジュールする                        |
| [micode](https://github.com/vtemian/micode)                                                               | 構造化されたブレインストーミング → 計画 → セッション継続性のあるワークフローの実装                                   |
| [octto](https://github.com/vtemian/octto)                                                                 | 複数の質問フォームを使用した AI ブレインストーミング用のインタラクティブなブラウザ UI                                |
| [nexode-background-agents](https://github.com/kdcokenny/nexode-background-agents)                     | 非同期委任とコンテキスト永続性を備えた Claude Code スタイルのバックグラウンドエージェント                            |
| [nexode-notify](https://github.com/kdcokenny/nexode-notify)                                           | Nexode のネイティブ OS 通知 – タスクがいつ完了したかを知る                                                         |
| [nexode-workspace](https://github.com/kdcokenny/nexode-workspace)                                     | バンドルされたマルチエージェントオーケストレーションハーネス – 16 コンポーネント、1 回のインストール                 |
| [nexode-worktree](https://github.com/kdcokenny/nexode-worktree)                                       | Nexode 用のゼロフリクション Git ワークツリー                                                                       |
