## Do Not Translate (Locale Additions)

- `Nexode` (preserve casing in prose; keep `nexode` only in commands, package names, paths, or code)
- `Nexode CLI`
- `CLI`, `TUI`, `MCP`, `OAuth`
- Commands, flags, file paths, and code literals (keep exactly as written)
